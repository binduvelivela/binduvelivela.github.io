App Name: Georgian-Connect
Role: Team member
Languages used: React-Native
About project: This application is designed for connecting students at Georgian college with other students. It is like a facebook for Georgian College only. Students can connect with each other from the same course, interest, and native country. As an international student in Georgian college, we have come to realize that it is very inconvenient not being able to connect with other Georgian students, especially in today’s circumstances where everybody is in comfort of their home and attending classes online. During these times most of the students opt to stay at home and study online. This has raised a concern towards students’ interactions with each other. Student networking was never an issue before COVID, as attending lectures, performing group activities, or just hanging out made it seem easy to connect with each other. The aim of this project is to enable students to find other Georgian students on the basis of course, location, interests, hometown etc. Students will be able to find other students who are pursuing the same course (senior/junior or classmates) as well as students with the same interests, same location, or hometown from other courses as well.
Initially, users will signup/login into the application. Once he is logged in will be redirected to the home screen and connect with other students. The “Home” screen has a navigation bar to see the latest notifications and navigate to other pages. Users can send friend requests to the students, while sending the request the user can see basic information and short bio of other users. The user can see each other's complete profile when they are friends. The user can edit their own personal details by going to “My profile” there is an edit button to edit the profile. The user can accept or decline the request from the friend request page which will be visible in notification. From the burger menu the person can change the password screen by entering old and new passwords for the same. From my profile the user can log out also. There is validation at each phase.
